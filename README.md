# Aggregate Data Exchange Mediator v2

## Description

NestJS port of Aggregate Data Exchange Mediator

## Config

1. copy .env.sample to .env
2. update relevant keys in .env file

## Installation

```bash
# install deps
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Docker

```bash
# init assembly
docker-compose up
```
